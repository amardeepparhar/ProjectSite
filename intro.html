Project Detailed Summary:

The proto-type site consists of the following pages:

  • Home
  • Lecture
  • Quiz
  • Example

The prototype was designed with usability and clarity in mind. The whole goal of this project is to redesign a existing BCIT site that is used regularly by students.

The goal of this new site is to provide easy access to notes, examples and other information. In addition another goal would be to upgrade the website to be a HTML5 and CSS3 compliant site.
